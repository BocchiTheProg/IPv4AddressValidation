Програма для перевірки якщо заданий аргумент строки є IPv4-адресою
Програма складається з двух методів які виконують однакову функцію:
- повертають значення true якщо задана строка є IPv4-адресою;
- повертають значення false якщо задана строка не є IPv4-адресою.
Один метод виконує перевірку лише за допомогою регулярного виразу, інший - за допомогою певної комбінації простих дій.
P.S. Методи приймають лише адреси в канонічному представленні, тобто без ведучих нулів, пропусків тощо.
